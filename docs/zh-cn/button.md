# Button

## 用法

Button 按钮组件

```html
<!-- 通过自定义类名，修改按钮样式-->
<wmui-button className="btn-info"></wmui-button>

<!-- 按钮禁用 -->
<wmui-button disabled></wmui-button>

```


## Props

| 参数 | 描述 | 类型 | 默认值 | 可选值 |
|------|-----|------|--------|-------|
| type  | 按钮类型 | String | button | reset/submit |
| className  | 自定义类名 | String | 无 | 无 |
| disabled | 是否禁用 | Boolean | false | true |
